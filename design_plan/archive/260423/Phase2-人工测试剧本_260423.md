## P1: Passive Trigger — 被动触发验证

**场景ID**: `P1`  
**目标**: 验证 SKILL.md 中的 PASSIVE TRIGGER 部分正确引导宿主 agent 在错误模式时建议 `/aristotle`  
**前置条件**: Aristotle 已作为 skill 安装到 Claude Code 或 OpenCode

### P1-A: Agent 自我纠正触发（模式 1）

**对应 SKILL.md 触发模式：** "You corrected your own output (acknowledged a mistake)"

**步骤：**
1. 要求 agent 实现一个有隐蔽 bug 的功能（例如 "Write a function to sort an array"）
2. Agent 产出代码后，不要用户干预，观察 agent 是否在后续思考中**自己发现**了问题
   - 或：在后续对话中让 agent review 自己的代码（例如 "Can you review the code you just wrote?"）
3. Agent 主动说类似 "Wait, there's an issue with..." 或 "Actually, I notice a bug..."
4. Agent 自行修正

**期望：** Agent 在自我纠正后输出被动触发建议：
> 🦉 I detected an error pattern. Run /aristotle to reflect and prevent similar mistakes.

**断言：**
- ✅ Agent **自己**发现错误（非用户指出）
- ✅ Agent 建议运行 `/aristotle`
- ✅ Agent **不**自动调用 `/aristotle`

### P1-B: 方案切换触发（模式 3）

**对应 SKILL.md 触发模式：** "You tried an approach, it failed, and you switched approaches"

**步骤：**
1. 给 agent 一个有挑战性的任务，使其可能尝试失败后换方案
2. Agent 尝试方案 A 但失败（例如编译错误、测试不通过）
3. Agent 说 "Let me try a different approach..." 并切换到方案 B

**期望：** Agent 在方案切换后输出被动触发建议

**断言：**
- ✅ Agent 主动切换方案（非用户指示）
- ✅ 触发被动建议

### P1-C: 用户指出错误触发（模式 2）

**对应 SKILL.md 触发模式：** "User pointed out an error and you agreed"

**步骤：**
1. 要求 agent 实现某功能
2. Agent 产出有错误的代码
3. **用户**明确指出错误："不对，这段代码没处理空数组" / "That's wrong, it doesn't handle empty arrays"
4. Agent 同意并修正

**期望：** Agent 在同意用户纠正后输出被动触发建议

**断言：**
- ✅ **用户**指出错误，Agent 同意
- ✅ 触发被动建议

### P1-D: 无误触发验证

**步骤：**
1. 正常对话（提问、回答）
2. 没有纠错或错误发生

**期望：** 不触发任何 Aristotle 建议

### P1-E: 思考阶段自我纠正（不触发 — 正确行为）

**步骤：**
1. Agent 在思考/推理阶段遇到错误
2. Agent 在内部认识到错误，但在输出最终结果前自行修正
3. 最终输出给用户的内容已经是正确的——对话中没有可见错误

**期望：** 不触发 Aristotle 建议

**合理性说明：** Agent 在错误到达对话之前已自行解决。Passive Trigger 监控的是可见的对话模式，而非内部推理状态。

### P1-F: 主会话纠正子代理错误（触发）

**步骤：**
1. 一个子代理（通过 `task()` 生成）返回了有错误的结果
2. 主会话 agent 审查子代理的输出
3. 主会话 agent 发现错误并予以修正

**期望：** 被动触发建议出现

**合理性说明：** 从主会话的角度看，这匹配模式 1（"You corrected your own output"）。多 agent 错误检测场景已被 SKILL.md 的触发模式显式覆盖。

## P2: 被动触发不应误触发

**场景ID**: `P2`  
**目标**: 验证正常操作不触发被动建议  
**前置条件**: 同 P1

以下对话**不应**触发 Aristotle 建议：
- 用户询问正常问题
- Agent 正常回答，无纠错
- 用户说 "谢谢" / "好的" / "继续"
