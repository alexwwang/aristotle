### TC-1-01: 完整 4 步状态转换 — init→reflecting→checking→done

- **测试函数名**: `test_full_reflect_flow`
- **所属类**: `TestOrchestrateOnEventReflect`
- **前置条件**: 
  1. `tmp_repo` fixture 生效
  2. `init_repo_tool()` 已调用（auto-init 内部会做）
- **输入**: 
  - Step 1: `orchestrate_start("reflect", {"target_session_id": "ses_target1", "target_label": "current"})`
  - Step 2: `orchestrate_on_event("subagent_done", {"workflow_id": "<wf_id>", "session_id": "ses_r_reflector", "result": "DRAFT persisted."})`
  - Step 3: `orchestrate_on_event("subagent_done", {"workflow_id": "<wf_id>", "session_id": "ses_c_checker", "result": "Committed: 2, Staged: 0"})`
- **预期输出**:
  - Step 1: `{action: "fire_sub", workflow_id: "wf_...", sub_prompt: "...", sub_role: "R", notify_message: "🦉 Aristotle Reflector launched..."}`
  - Step 2: `{action: "fire_sub", workflow_id: "wf_...", sub_prompt: "...", sub_role: "C"}`
  - Step 3: `{action: "notify", workflow_id: "wf_...", message: "🦉 Aristotle done. 2 rules committed, 0 staged.\n   Review: /aristotle review 1"}`
- **断言**:
  ```python
  # Step 1 assertions
  assert start["action"] == "fire_sub"
  assert start["sub_role"] == "R"
  assert start["workflow_id"].startswith("wf_")
  assert "notify_message" in start
  assert "Reflector launched" in start["notify_message"]
  wf_id = start["workflow_id"]
  
  # Verify workflow state after start
  wf = _load_workflow(wf_id)
  assert wf["phase"] == "reflecting"
  assert wf["command"] == "reflect"
  assert wf["pending_role"] == "R"
  assert wf["record_created"] is False
  assert wf["sequence"] == 1
  assert wf["target_session_id"] == "ses_target1"
  
  # Step 2 assertions (R done → fire C)
  assert r_done["action"] == "fire_sub"
  assert r_done["sub_role"] == "C"
  assert r_done["workflow_id"] == wf_id
  
  # Verify workflow state after R done
  wf = _load_workflow(wf_id)
  assert wf["phase"] == "checking"
  assert wf["pending_role"] == "C"
  assert wf["record_created"] is True
  
  # Step 3 assertions (C done → notify)
  assert c_done["action"] == "notify"
  assert "committed" in c_done["message"]
  assert "/aristotle review" in c_done["message"]
  assert "2 rules committed" in c_done["message"]
  
  # Verify workflow state after C done
  wf = _load_workflow(wf_id)
  assert wf["phase"] == "done"
  ```
- **覆盖的验收标准**: A4, A8
- **状态机路径**: init → reflecting → checking → done

### TC-1-02: record_created 标记从 False→True（R 完成时创建 record）

- **测试函数名**: `test_r_done_creates_reflection_record`
- **所属类**: `TestOrchestrateOnEventReflect`
- **前置条件**:
  1. `tmp_repo` 生效
  2. 已调用 `orchestrate_start("reflect", {"target_session_id": "ses_tgt"})` 获得 `wf_id`
- **输入**:
  - `orchestrate_on_event("subagent_done", {"workflow_id": wf_id, "session_id": "ses_r_special", "result": "DRAFT persisted to: /tmp/rec_1.md"})`
- **预期输出**:
  - `{action: "fire_sub", sub_role: "C", ...}`
  - `aristotle-state.json` 新增一条 record，含 `reflector_session_id: "ses_r_special"`
- **断言**:
  ```python
  # 1. record_created 从 False 变为 True
  wf_before = _load_workflow(wf_id)
  assert wf_before["record_created"] is False
  
  result = _fire_r_done_event(wf_id, session_id="ses_r_special")
  
  wf_after = _load_workflow(wf_id)
  assert wf_after["record_created"] is True
  
  # 2. state file 新增 record
  from aristotle_mcp.config import resolve_repo_dir
  state_path = resolve_repo_dir().parent / "aristotle-state.json"
  assert state_path.exists()
  records = json.loads(state_path.read_text(encoding="utf-8"))
  assert len(records) >= 1
  assert records[-1]["reflector_session_id"] == "ses_r_special"
  assert records[-1]["target_session_id"] == "ses_tgt"
  assert records[-1]["status"] == "processing"
  
  # 3. 返回 fire_sub(C)
  assert result["action"] == "fire_sub"
  assert result["sub_role"] == "C"
  ```
- **覆盖的验收标准**: A4
- **状态机路径**: reflecting → checking（record_created: False→True）

### TC-1-04: sequence 递增 — 连续两次 reflect 产生不同 sequence

- **测试函数名**: `test_reflect_sequence_incremented`
- **所属类**: `TestOrchestrateStartReflect`
- **前置条件**:
  1. `tmp_repo` 生效
  2. 空 `aristotle-state.json`
  3. 第 1 次 reflect 的 R 已完成（写入 state record）
- **输入**:
  - 第 1 次: `orchestrate_start("reflect", {"target_session_id": "ses_1"})` → R done → C done
  - 第 2 次: `orchestrate_start("reflect", {"target_session_id": "ses_2"})`
- **预期输出**:
  - 第 1 次: `sequence=1`，C done 后 state record 写入
  - 第 2 次: `sequence=2`（从 state record 计数）
- **断言**:
  ```python
  r1 = _start_reflect_workflow("ses_1")
  wf1_id = r1["workflow_id"]
  wf1 = _load_workflow(wf1_id)
  assert wf1["sequence"] == 1
  
  # 完成 R+C 才会写入 state record
  _fire_r_done_event(wf1_id, "ses_r1")
  _fire_c_done_event(wf1_id, "Committed: 0, Staged: 0")
  
  # 第 2 次 reflect，_next_sequence() 从 state file 计算
  r2 = _start_reflect_workflow("ses_2")
  wf2_id = r2["workflow_id"]
  wf2 = _load_workflow(wf2_id)
  assert wf2["sequence"] == 2
  
  # workflow_id 不同
  assert wf1_id != wf2_id
  ```
- **覆盖的验收标准**: A4
- **状态机路径**: init→reflecting→checking→done, init→reflecting

### TC-1-06: C 完成 partial_commit — staged>0 时状态为 partial_commit

- **测试函数名**: `test_c_done_partial_commit_status`
- **所属类**: `TestOrchestrateOnEventReflect`
- **前置条件**:
  1. `tmp_repo` 生效
  2. reflect workflow 处于 `phase=checking`（已完成 R done）
- **输入**:
  - `orchestrate_on_event("subagent_done", {workflow_id, result: "Committed: 1, Staged: 2"})`
- **预期输出**:
  - `{action: "notify", message: "🦉 Aristotle done. 1 rules committed, 2 staged."}`

- **断言**:

  ```python
  start = _start_reflect_workflow("ses_tgt")

  assert records[-1]["rules_count"] == 3

  assert wf["phase"] == "done"
  ```
